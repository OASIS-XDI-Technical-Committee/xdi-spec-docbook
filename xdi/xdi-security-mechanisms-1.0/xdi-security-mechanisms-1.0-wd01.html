Abstract:

This Working Draft 01 addresses operational and protocol security requirements for the storage and transport of XDI graph information as a set of conformance profiles. In addition, it defines a template for the creation of additional profiles.

1.2.1 Key words

The key words MUST, MUST NOT, REQUIRED, SHALL, SHALL NOT, SHOULD, SHOULD NOT, RECOMMENDED, MAY, and OPTIONAL are to be interpreted as described in [RFC 2119].

1.2.2 Glossary of Terms

XDI

XRI Data Interchange

Normalized Graph

the canonical form of the graph, necessary to ensure successful signature validation.

Signature Block

a collection of XDI statements describing metadata about the signature sufficient to ensure successful validation, and the signature itself.
